**Midas** - настраиваемый экономический плагин для серверов **Allay**. Он добавляет магазин через формы, скупщика через формы, обновляемые особые предложения и личные авто-скупщики, привязанные к сундукам или бочкам.

### ✨ Возможности
- **Магазин через формы**: Категории и товары настраиваются в `shop.yml`.
- **Гибкие награды**: Товар может выдавать предмет, выполнять команду или накладывать эффект.
- **Скупщик**: Игроки продают настроенные предметы через `/buyer`.
- **Выбор количества**: Цена считается пропорционально количеству или фиксируется настройкой товара.
- **Особые предложения**: Скидки в магазине и повышенная цена продажи у скупщика.
- **Фиксированные спецтовары**: Отдельные товары с ценами, заранее указанными в конфиге.
- **Авто-скупщик**: Привязка сундука или бочки и продажа подходящих предметов через личную форму контейнера.
- **Локализация**: Тексты лежат в `assets/lang/en_US.json` и `assets/lang/ru_RU.json`.

### 🎮 Команды
| Команда | Право | Описание |
|---------|-------|----------|
| `/shop` | `midas.command.shop` | Открыть форму магазина. |
| `/buyer` | `midas.command.buyer` | Открыть форму скупщика. |
| `/setautobuyer` | `midas.command.autobuyer` | Привязать сундук или бочку авто-скупщика. |
| `/delautobuyer` | `midas.command.autobuyer.delete` | Отвязать сундук или бочку авто-скупщика. |

### ⚙️ Конфигурация
Midas создает два основных YAML-файла в папке данных плагина:

| Файл | Описание |
|------|----------|
| `shop.yml` | Заголовок магазина, валюта, категории, товары и особые предложения магазина. |
| `buyer.yml` | Заголовок скупщика, категории продажи, особые предложения скупщика и авто-скупщики. |

### 📝 Настройки товара
| Поле | Описание |
|------|----------|
| `id` | Уникальный ID товара внутри категории. |
| `name` | Название в форме. |
| `description` | Описание в форме. |
| `minAmount` | Минимальное количество для покупки или продажи. |
| `price` | Цена за `minAmount`. |
| `amountScrollEnabled` | Если `true`, игрок выбирает количество; если `false`, используется фиксированное количество. |
| `maxAmount` | Максимальное количество при включенном выборе. |

Товары магазина поддерживают награды `ITEM`, `COMMAND` и `EFFECT`. Товары скупщика используют фильтр предмета: материал, data, имя, lore и зачарования.

### 🔥 Особые предложения
Особые предложения магазина обновляются через заданный интервал и включают:
- случайные обычные товары со скидкой от `discountMinPercent` до `discountMaxPercent`;
- настроенные спецтовары с фиксированной ценой.

Особые предложения скупщика работают аналогично и включают:
- случайные обычные товары с множителем цены от `multiplierMin` до `multiplierMax`;
- настроенные спецтовары с фиксированной ценой.

### 📦 Установка
1. Соберите или скачайте jar-файл плагина.
2. Положите его в папку `plugins` сервера.
3. Установите обязательные экономические зависимости.
4. Перезапустите сервер.
5. При необходимости отредактируйте `shop.yml` и `buyer.yml`.

### 🛠️ Сборка
```bash
./gradlew build
```

Готовый jar будет в `build/libs`.

### 🔗 Зависимости
- **Обязательно**: Allay Server `0.27.0` или новее.
- **Обязательно**: EconomyAPI `0.2.2` или новее.
- **Обязательно**: Aconomy `0.2.0` или новее.
